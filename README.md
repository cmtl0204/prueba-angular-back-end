### Requisitos
- PHP 7.4
- Composer
- Apache o Nginx
- Postgresql

### Iniciar
- composer install
- Crear una copia de .env.example y cambiar el nombre a .env
- Configurar en el archivo .env las credenciales de la base de datos
- Ejecutar en la consola "php artisan migrate:fresh --seed"
